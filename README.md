# autoDingTalk
钉钉自动打卡脚本

## install
1. auto.js
https://github.com/SuperMonster003/AutoJs6
2. 克隆本仓库
```
git clone https://github.com/goxixi/autoDingTalk.git
``` 
3. 根据自己手机修改autoDingTalk.js脚本
①截图功能

4. 设置定时脚本
在安卓端auto.js添加两个脚本，分别设置定时时间为上班时间和下班时间
